Program Overview:

1)The following java program calculates the number of tickets and ticket prices 
for ticket types-A,B,C.
2) The input is taken from the user

Number of methods:3
calculatePriceTicketA()
calculatePriceTicketB()
calculatePriceTicketC()

Exceptions handled:
1) negative number exception
2) inputformat mismatch exception
3) invalid ticket prices exception

Logic used:
1) calculate ticket price= numberOfTickets*priceOfTicket
2) calculate the over all price= sum of ticket prices of (A+B+C)
